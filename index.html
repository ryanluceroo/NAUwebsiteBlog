Courses That I have taken

A brief summary of some of the computer science related courses I have taken at NAU.

Intoduction to Computer Organization

This class is structured to touch on Binary representation of information in digital computers. It also includes an introduction to digital logic design, computer architectures, microprocessor architectures and assembly language programming.

Computer Science 2

Computer Science II is an Intermediate computer science class that emphasizes in algorithm design, object-oriented programming, and data structures.

Computing Tools 1 ,2 ,3

Computing Tools is a series of classes that cover critical developmental tools that are often used in the computer science, informatics, and engineering industries. Some of the topics include command line tools, shell scripting, git or project management tools and deploying remote services
